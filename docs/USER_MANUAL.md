# Smart Citizen User Manual

## Citizen workflow

1. Open the portal and log in or create a citizen account.
2. Use **Submit Complaint** to describe the problem.
3. Add location, phone, evidence file, evidence link, or voice note when needed.
4. Submit the complaint and keep the tracking number.
5. Use **My Complaints** or public tracking to follow progress.
6. Read official responses from the responsible office.
7. Rate the resolution when the case is marked resolved.

## Staff workflow

1. Log in with an Administrative Staff account.
2. Open **Cases**.
3. Review cases assigned to your office.
4. Use **Start reviewing**, **Ask the citizen for more**, **Escalate**, or write an official answer and **Mark as resolved**.
5. Use **Reports** to see office workload and status counts.

## Admin workflow

1. Log in with an Admin account.
2. Open **Overview** for sector-level totals.
3. Open **All Complaints** to filter and inspect all cases.
4. Open **Categories, SLA & Routing** to configure which office receives each complaint type.
5. Open **Users & Roles** to manage accounts.
6. Open **Reports** to print or download official complaint reports.
7. Open **Audit Log** to review actions taken in the system.
